# TaipeiMetroDraw｜雙北捷運抽站

出遊團的隨機探訪工具：**先隨機抽一條捷運線，再從該線抽出三個站**，然後——出發去探訪！

🔗 **線上使用**：<https://a102304043.github.io/TaipeiMetroDraw/>

## 特色

- 手刻 SVG 雙北捷運路網示意圖，抽選後自動 zoom 到最合適視野、高亮抽中的三個站
- 抽選池涵蓋 10 條線、192 站（2026-07 營運現況）：北捷六線、環狀線、淡海輕軌、安坑輕軌、**三鶯線（試營運）**、桃園機場捷運全線
- 抽選歷史與「排除去過的站」模式（localStorage，資料不出裝置）
- 單一 `index.html`，零依賴、零 build，開網頁就能用

## 技術備忘

- 純 vanilla HTML/CSS/JS 單檔，部署於 GitHub Pages
- 地圖座標系：虛擬畫布 1500×1800、網格 U=50、45° octilinear
- 開發用工具：網址加 `?debug=labels` 顯示站名 label 重疊偵測
