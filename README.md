# TaipeiMetroDraw｜雙北捷運抽站

出遊團的隨機探訪工具：**先隨機抽一條捷運線，再從該線抽出三個站**，然後——出發去探訪！

🔗 **線上使用**：<https://a102304043.github.io/TaipeiMetroDraw/>

## 特色

- 🗺️ 手刻 SVG 雙北捷運路網示意圖（官方圖風格、45° 折線），抽選後自動 zoom 到最合適視野
- 🎰 抽選儀式：老虎機抽線 → 鏡頭推進 → 光點沿線跳選三站 → 編號標記 ①②③（螢幕尺寸補償，長線全景也清楚）
- 🚇 抽選池 10 條線、169 站（2026-07 營運現況）：北捷六線、環狀線、淡海輕軌、安坑輕軌、**三鶯線（試營運）**、桃園機場捷運全線
- 📇 車站資訊卡：維基百科照片＋簡介（CC BY-SA）、行政區與轉乘、每日探訪任務籤、一鍵導航
- 🎨 主題：經典路網（深淺色）／深夜霓虹；音效（WebAudio 合成，預設關）；震動回饋（Android）
- 📝 抽選歷史＋「已探訪」收集進度＋排除去過的站／排除機捷桃園段
- 📦 單一 `index.html`，零依賴、零 build；資料只存在你的裝置（localStorage）

## 使用備忘

- 地圖可雙指縮放／拖曳，右下 ⌖ 回到抽選結果視野
- 點地圖上任何站或結果籤 → 開車站資訊卡
- 抽選動畫期間點畫面任意處可跳過
- 廣慈/奉天宮站（R01）已內建，正式通車後改 `enabled: true` 即上架

## 技術備忘

- 純 vanilla HTML/CSS/JS 單檔，部署於 GitHub Pages
- 地圖座標系：網格 U=50、45° octilinear、座標吸附 25；相機為 `{cx, cy, scale}` 單一狀態＋rAF viewBox 插值
- Label 三級 LOD（全景不畫站名 → 中距轉乘＋端點 → 近距全部），相機動畫期間鎖定目標層級（手機效能關鍵）
- 開發工具：
  - `?debug=labels`——站名 label 重疊偵測
  - `node tools/verify-data.mjs`——站名／順序／octilinear 幾何／地理方位驗證
  - `node tools/verify-wiki.mjs`——逐站解析維基條目（消歧義），產出 `WIKI_TITLES` 覆寫表
